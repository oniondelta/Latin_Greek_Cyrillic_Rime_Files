#### ※ Commercial use is prohibited

## 方案簡介

- 4in1

  > 匯集以下，其所有字母皆可打出
  
  > 26個英文字母直出，其他字母啟始鍵：「 ; 」「 ' 」「 ] 」「 [ 」
  
  > 會用到 opencc 內檔案，需置入。

- Latin

  > Unicode 拉丁語系字母，法文、德文、捷克文、越南文…等。如：(ß, å, ó, ằ, Ᵹ)

  > Unicode 所有音標，IPA國際音標、DJ音標、KK音標…等。如：( ː , ɔ, ǝ, ʒ, ʤ, ʥ, ŋ, ð, a͡e)

  > 會用到 opencc 內檔案，需置入。

- Cyrillic

  > Unicode 西里爾語系字母，俄文…等。如：(д, П, Ж )

- Greek

  > Unicode 希臘語系字母，希臘文…等。如：(Δ, π, ε )

- Fullshape

  > 全形 26 個字母和相關全形符號
